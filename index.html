Information

This tool generates a side-by-side bilingual edition of any text you give it. Pick the input that matches what you have.

Paste text

Drop any passage into the source box and the generator splits it at full stops, translates each sentence, and packages the pairs into a single-chapter EPUB. Best for short stories, articles, or anything you can copy out of a webpage or PDF — the bilingual output is one short pair per sentence, not per paragraph. Use the sample passage if you want to see the expected format before pasting your own text.

Upload EPUB

Pick an existing .epub file and the generator parses its spine, walks every chapter's paragraphs and headings, translates each block, and assembles a new bilingual EPUB that preserves chapter boundaries, the original chapter titles, and the cover image. Best for full books. The source language and book title are detected from the EPUB metadata, so most of the time you only need to fill in the target language.

Saving as EPUB

When generation finishes, a Download EPUB button appears at the top of the page. Click it to save the bilingual edition as a standard EPUB you can open in any reader.

Saving as PDF

If you'd rather have a PDF, once the bilingual book is on screen press Ctrl + P (Windows) or Command + P (Mac), choose Save as PDF, and set margins to 0 for the cleanest layout.
